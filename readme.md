# **Evaluación Final · Módulo 3 · eaguilothio**

## **Descripción**
Proyecto de análisis de datos orientado a comprender el perfil de los clientes y su comportamiento dentro de un programa de fidelización de una aerolínea.  
El objetivo es **explorar, limpiar, integrar y analizar** la información para extraer conclusiones relevantes a partir de datos.

---

## **Fuentes de datos**

El análisis se basa en dos datasets complementarios:

### **1. Customer Flight Activity.csv**
Contiene información mensual sobre la actividad de vuelo y uso de puntos, tales como:
- Vuelos reservados  
- Distancia volada  
- Puntos acumulados y redimidos  
- Coste en dólares de los puntos redimidos  

### **2. Customer Loyalty History.csv**
Incluye datos demográficos y de fidelización del cliente, tales como:
- Ubicación (país, provincia, ciudad)  
- Género  
- Nivel educativo  
- Salario  
- Tipo de tarjeta de fidelidad  

---

## Objetivo
Obtener insights relevantes y reflexiones en cada fase del proceso de análisis de datos (exploración, limpieza, integración y análisis) para comprender el comportamiento de los clientes y fundamentar las conclusiones.

---

## Aspectos trabajados 
- Exploración inicial para identificar patrones y posibles inconsistencias.  
- Limpieza y preparación de la información para su análisis.  
- Integración de los distintos conjuntos de datos.  
- Análisis de variables clave relacionadas con el comportamiento del cliente.  
- Representación gráfica de los principales resultados.  
- Interpretación de resultados.

---

## Herramientas utilizadas
- `pandas`, `numpy` para el tratamiento y análisis de datos.  
- `matplotlib`, `seaborn` para la visualización de resultados.
